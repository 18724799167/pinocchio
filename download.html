The software architecture has been tested entirely on Ubuntu 10.04 LTS, 12.04 LTS, and 14.04 LTS. If you are running other distributions, we expect the library to work correctly but we can not guarantee it.
To install Pinocchio from sources, you will first need to perform installations of other packages themselves dependent on other ones. To facilitate the procedure, you will find in the part "Build From Source" all you need to install Pinocchio. Pinocchio comes with a python binding that you can use to prototype and that makes the link with Gepetto-viewer to vizualise your robot ( Gepetto viewer is a simple 3D-vizualisation tool we advice)
